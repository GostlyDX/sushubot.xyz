Aqui tienes la documentación del bot, esta documentación te mostrara los comandos que hay y el uso de cada uno de ellos! ^^

Comandos :

• avatar

• ban

• cagarloapalos

• help

• kick

• maintenance

• meme

• ping

• purge

• restaurarvenezuela

• say

Avatar :

Este comando mostrara la foto de perfil del usuario al que menciones despues de hacer uso del comando!

Uso de este comando : s/avatar {usuario}

Ban :

Este comando baneara al usuario que se mencione despues de hacer uso del comando!

Uso de este comando : s/ban {usuario} {razón}

Cagarloapalos :

Con este comando golpearas a alguien con un palo!

Uso de este comando : s/cagarloapalos {usuario}

Help :

Con este comando recibiras información del bot!

Uso de este comando : s/help

Kick :

Este comando expulsara al usuario mencionado despues de el comando!

Uso de este comando : s/kick {usuario} {razón}

Maintenance :

Este comando te mostrara los comandos en mantenimiento!

Uso de este comando : s/maintenance

Meme :

Este comando te mostrara un meme aleatorio!

Uso de este comando : s/meme

Ping :

Este comando te mostrara el MS del bot!

Uso de este comando : s/ping

Purge :

Este comando borrara un máximo de 99 mensajes!

Uso de este comando : s/purge {cantidad}

Restaurarvenezuela :

Con este comando podras restaurar Venezuela!

Uso de este comando : s/restaurarvenezuela

Say :

Con este comando podras hacer que el bot diga lo que tu quieras!

Uso de este comando : s/say {texto}
